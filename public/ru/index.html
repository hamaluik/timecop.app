Time Cop

Приложение для отслеживания времени, которое уважает вашу конфиденциальность и выполняет свою работу, не становясь слишком модным.

Характеристики

  • Только в автономном режиме, только для мобильных устройств (iOS / Android)
  • Полностью приватный - нет слежки / шпионажа / рекламы / и т.д.
  • Отслеживайте задачи с несколькими параллельными таймерами, которые можно запустить одним нажатием кнопки
  • Свяжите таймеры с проектами, чтобы сгруппировать вашу работу (или нет)
  • Запускать, останавливать, редактировать и удалять таймеры всякий раз, когда без суеты
  • Экспорт данных в виде файла .csv , отфильтрованный по временным интервалам и проектам
  • Экспорт базы данных приложения для полного доступа ко всем его данным
  • Автоматический светлый режим / темный режим в зависимости от настроек вашего устройства
  • Локализовано на нескольких языках (благодаря Google Translate ): английский, арабский, немецкий, испанский, французский, хинди, индонезийский, японский, корейский, португальский, русский, китайский (упрощенный), китайский (традиционный)
  • Открытый исходный код ( лицензируется под Apache-2.0 ) - забывайте!
