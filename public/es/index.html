Time Cop

Una aplicación de seguimiento del tiempo que respeta su privacidad y hace el trabajo sin ser demasiado elegante.

Caracteristicas

  • Sin conexión, solo para dispositivos móviles (iOS / Android)
  • Totalmente privado: no hay seguimiento / espionaje / publicidad / etc.
  • Realice un seguimiento de las tareas con varios temporizadores paralelos que se pueden iniciar con solo tocar un botón
  • Asocia los temporizadores con proyectos para agrupar tu trabajo (o no)
  • Inicie, detenga, edite y elimine temporizadores siempre que no tenga problemas
  • Exportar datos como un archivo .csv , filtrado por períodos de tiempo y proyectos
  • Exporte la base de datos de la aplicación para tener acceso completo a todos sus datos.
  • Modo automático claro / oscuro basado en la configuración de su dispositivo
  • Localizado en varios idiomas (gracias a Google Translate ): inglés, árabe, alemán, español, francés, hindi, indonesio, japonés, coreano, portugués, ruso, chino (simplificado), chino (tradicional)
  • Código abierto ( licenciado bajo Apache-2.0 ): ¡bifurca!
