Time Cop

Eine Zeiterfassungs-App, die Ihre Privatsphäre respektiert und die Arbeit erledigt, ohne zu ausgefallen zu werden.

Eigenschaften

  • Nur offline, nur mobil (iOS / Android)
  • Völlig privat - es gibt keine Verfolgung / Spionage / Werbung / etc.
  • Verfolgen Sie Aufgaben mit mehreren parallelen Timern, die per Knopfdruck gestartet werden können
  • Verknüpfen Sie Timer mit Projekten, um Ihre Arbeit zu gruppieren (oder nicht).
  • Starten, stoppen, bearbeiten und löschen Sie Timer immer dann, wenn Sie keine Probleme haben
  • Exportieren Sie Daten als .csv Datei, gefiltert nach Zeiträumen und Projekten
  • Exportieren Sie die Datenbank der App, um vollen Zugriff auf alle Daten zu erhalten
  • Automatischer Hell- / Dunkelmodus basierend auf Ihren Geräteeinstellungen
  • In mehreren Sprachen lokalisiert (dank Google Translate ): Englisch, Arabisch, Deutsch, Spanisch, Französisch, Hindi, Indonesisch, Japanisch, Koreanisch, Portugiesisch, Russisch, Chinesisch (vereinfacht), Chinesisch (traditionell)
  • Open Source ( lizenziert unter Apache-2.0 ) - weggabeln!
