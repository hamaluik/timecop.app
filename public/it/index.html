Time Cop

Un'app di monitoraggio del tempo che rispetta la tua privacy e fa il lavoro senza essere troppo fantasioso.

Caratteristiche

  • Solo offline, solo mobile (iOS / Android)
  • Completamente privato: non esiste tracciamento / spionaggio / pubblicità / ecc
  • Tieni traccia delle attività con più timer paralleli che possono essere avviati con il tocco di un pulsante
  • Associa i timer ai progetti per raggruppare il tuo lavoro (o no)
  • Avvia, ferma, modifica ed elimina i timer ogni volta che non hai problemi
  • Esporta i dati come file .csv , filtrati per timespan e progetti
  • Esporta il database dell'app per avere pieno accesso a tutti i suoi dati
  • Modalità luce automatica / modalità oscura in base alle impostazioni del dispositivo
  • Localizzato in diverse lingue (grazie a Google Translate ): inglese, arabo, tedesco, spagnolo, francese, hindi, indonesiano, giapponese, coreano, portoghese, russo, cinese (semplificato), cinese (tradizionale)
  • Open source ( concesso in licenza con Apache-2.0 ) —fork away!
