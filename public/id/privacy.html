Kebijakan Privasi Time Cop

Saya tidak menyimpan atau mengakses data Anda . Saya tidak bisa, dengan desain. Saya tidak menggunakan layanan pihak ketiga mana pun yang menyimpan atau mengakses data Anda (terlepas dari toko aplikasi tempat Anda mengunduh aplikasi tersebut, tetapi saya juga tidak mengakses data dari sana).

Time Cop sengaja tidak memasukkan kode pelacakan, kode iklan, atau bahkan kode pelaporan bug. Itu tidak ada di sana — Anda dapat memeriksa kode sumber untuk diri sendiri jika Anda tidak percaya, atau bahkan melangkah lebih jauh dan kompilasi sumber dan menjalankan biner tanpa pernah menyentuh binari saya. Saya tidak mengumpulkan alamat email Anda, atau data lainnya. Satu-satunya data yang tersedia untuk aplikasi Time Cop adalah data yang Anda masukkan sendiri secara langsung, atau hasilkan melalui penggunaan aplikasi.

Time Cop mengumpulkan dan menghasilkan data berikut, dan hanya data berikut:

  • Nama Proyek
  • Warna Proyek
  • Deskripsi Pengatur Waktu
  • Proyek Pengatur Waktu
  • Timer Mulai Waktu
  • Timer Akhir Waktu

Semua data yang dikumpulkan dan dihasilkan oleh aplikasi Time Cop tetap menjadi basis data di perangkat Anda sampai Anda mengekspor data secara manual, atau sebagiannya — dalam hal ini, data yang Anda pilih untuk ekspor akan disajikan ke ponsel sistem operasi untuk "dibagikan" dengan aplikasi apa pun yang Anda pilih. Pada titik mana pun dalam proses ini, saya, atau pihak ketiga mana pun, tidak memiliki akses ke data ini, dengan pengecualian sistem operasi host dan aplikasi pihak ketiga yang Anda pilih untuk mengekspor data.

Kebijakan ini dapat berubah, namun jika terjadi perubahan, aplikasi Time Cop akan memberi tahu Anda tentang:

  1. Perubahan yang dimaksud
  2. Alasan untuk perubahan yang dimaksud
  3. Tanggal ketika perubahan yang diinginkan akan mulai berlaku

Saya membuat Time Cop di waktu luang saya untuk penggunaan pribadi saya. Saya merasa cukup berguna sehingga saya percaya orang lain mungkin menganggapnya bernilai, dan karenanya saya menjadikannya aplikasi publik alih-alih sesuatu yang baru saja saya jalankan di perangkat saya sendiri. Ini tidak mengubah proposisi nilai aplikasi untuk saya sendiri. Jika saya perhatikan bahwa sejumlah instalasi telah terjadi di toko aplikasi, itu akan membuat saya senang bahwa orang lain menganggapnya berguna, tetapi itu tidak akan mengubah cara saya memperlakukan aplikasi dan data yang digunakannya.
