Time Cop

Aplikasi pelacakan waktu yang menghormati privasi Anda dan menyelesaikan pekerjaan tanpa menjadi terlalu mewah.

fitur

  • Khusus offline, khusus seluler (iOS / Android)
  • Sepenuhnya pribadi — tidak ada pelacakan / pengintai / iklan / dll
  • Melacak tugas dengan banyak penghitung waktu paralel yang dapat dimulai dengan ketukan tombol
  • Kaitkan penghitung waktu dengan proyek untuk mengelompokkan pekerjaan Anda (atau tidak)
  • Mulai, hentikan, edit, dan hapus penghitung waktu kapan pun tanpa repot
  • Mengekspor data sebagai file .csv , difilter menurut rentang waktu dan proyek
  • Ekspor database aplikasi untuk akses penuh ke semua datanya
  • Mode lampu otomatis / mode gelap berdasarkan pengaturan perangkat Anda
  • Lokal dalam beberapa bahasa (terima kasih ke Google Translate ): Inggris, Arab, Jerman, Spanyol, Prancis, Hindi, Indonesia, Jepang, Korea, Portugis, Rusia, Mandarin (Sederhana), Mandarin (Tradisional)
  • Open source ( berlisensi di bawah Apache-2.0 ) —baik saja!
